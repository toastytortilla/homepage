Projects

In no particular order, these are the projects that I'm most proud of:

Since we're here, we should start here! This website has been a fun project for me. The goal was to make a personal website. The most difficult part was navigating the intimdating vastness of the web dev world. To overcome this difficulty, I had to accept that my website won't (and shouldn't) be perfect on the first try and push myself to just learn new concepts one at a time.

Recete was my submission as my final project for CS50. The objective was to build an entire application from the ground up. Back end, front end, and everything in between was my responsibility. While conceptualizing, planning, creating, and hosting this application were definitely challenging, I also introduced some self-imposed difficulties by making the decision to completely divest myself from the CS50 library for the essential functions of the app. This meant that I was forced to learn the ins and outs of the Flask framework, SQLite module, and various other libraries in order to make the app fully functional in it's own right. Recover was a landmark assignment for me. The objective was to write a program in C that would recover 50 deleted .jpg photos from a .raw file. It was a challenge to understand how to read, write, and free memory in C. I'm extra proud of this assignment because it was the first one where I felt that I truly understood the required logic, it was proof that I was making progress.

Readability was suprisingly frustrating at first... The goal was to reconstruct a program we had written in C which would take text as input and return the approximate grade level of difficulty. Despite knowing the logic for the program and having some prior experience with Python, I quickly found that things weren't as straightforward as I assumed. Python can be powerful because of its abstraction, but it is still crucial to understand what is happening underneath the hood to make a well-designed program. Once I grasped what implicit assumptions Python and it's libraries were making, I was able to complete the project with relative ease.
